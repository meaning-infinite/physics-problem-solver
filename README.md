## 特性

- ✅ **不仅给答案，更给思路** - 说明解题思路和物理模型选择依据
- ✅ **关键推导** - 保留关键推导步骤，省略平凡的代数运算
- ✅ **路径清晰** - 从已知条件到最终结论，路径一目了然
- ✅ **多解列出** - 如果题目有多解，清晰列出所有可能情况
- ✅ **公式规范** - 统一使用 LaTeX 语法，支持行内公式、独立公式和带编号公式
- ✅ **验证机制** - 内部验证量纲、数量级、边界条件

## 使用示例

### 示例 1：自由落体

**题目**：质量为 $m$ 的物体从高度 $h$ 自由落下，求落地时的速度。

**已知**：$m$, $h$, $g$

**求**：$v$

**思路**：自由落体，机械能守恒。势能转化为动能。

**Step 1**：机械能守恒

$$
mgh = \frac{1}{2}mv^2 \tag{1}
$$

**Step 2**：解方程（省略代数运算）

$$
v = \sqrt{2gh} \tag{2}
$$

**答案**：$v = \sqrt{2gh}$

---

### 示例 2：并联电阻

**题目**：电阻 $R_1$ 和 $R_2$ 并联，求等效电阻。

**已知**：$R_1$, $R_2$

**求**：$R_{eq}$

**思路**：并联电阻，电压相同，电流相加。

**Step 1**：并联公式

$$
\frac{1}{R_{eq}} = \frac{1}{R_1} + \frac{1}{R_2} \tag{1}
$$

**Step 2**：解方程

$$
R_{eq} = \frac{R_1 R_2}{R_1 + R_2} \tag{2}
$$

**答案**：$R_{eq} = \frac{R_1 R_2}{R_1 + R_2}$

**特殊情况**：
- $R_1 = R_2 = R$：$R_{eq} = \frac{R}{2}$
- $R_1 \ll R_2$：$R_{eq} \approx R_1$

## 强度级别

| 级别 | 说明 |
|-----|------|
| **标准** | 完整步骤 + 思路说明 |
| **简洁** | 关键步骤 + 思路（省略更多推导） |
| **极简** | 核心方程 + 答案（仅简单问题） |

切换：`/physics 标准|简洁|极简`

## 核心原则

```
不说废话，直接解题。保留关键推导，省略平凡运算。路径一目了然。
```

## 解题流程

```
1. 审题 → 提取已知量、未知量、约束条件
2. 分析 → 选择物理模型、列出关键方程
3. 求解 → 代入数据、逐步推导（省略平凡代数运算）
4. 验证 → [内部验证，默认不输出] 仅在异常或要求时展示
5. 结论 → 简洁陈述最终答案，多解时列出所有情况
```
